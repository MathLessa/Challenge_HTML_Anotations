Introdução ao HTML na prática

Tags

As tags são os principais objetos estruturais do HTML, com elas conseguimos formatar os nossos textos, adicionar imagens, linkar outro site, etc. Veremos a seguir as tags mais comuns e utilizadas, assim como suas funções:

  • i: Formata o texto em itálico
  • strong: Formata o texto em negrito
  • input: Com essa tag conseguimos adicionar uma área de input para o usuário, sendo essa área um botão, uma caixa de texto, etc.
  • img: Adiciona uma imagem ao site utilizando o link ou localização da pasta no computador.
  • blockquote: Realiza uma citação no texto.
  • u: Coloca um sublinhado abaixo da palavra.
  • mark: Marca a palavra com uma cor de fundo.
  • sup: Coloca uma palavra menor e em posição superior.Ex: I2
  • sub: Coloca uma palavra menor e em posição inferior.Ex: Iequivalente
  • ul: Cria uma lista não ordenada.
  • ol: Cria uma lista ordenada.
  • li: Demarca os itens da lista.
  • a: Linka o elemento com algum site ou até mesmo outro HTML. Ex: No Módulo anterior foi mostrada um pouco da história dos computadores.
  • form: Cria um formulário de informações.

Atributos

Os atributos são modificadores das tags, utilizamos eles para adicionar funções ou características para a tag.

  • id: É o identificador, pode ser atrelada a uma tag para posteriormente ser identificada pelo JavaScript.
  • style: É utilizado para adicionar estilos e comandos do CSS diretamente no elemento.
  • class: Também atrelado ao CSS, é criada uma classe à tag, assim qualquer mudança de estilo realizada no css pode mudar em todos os elementos da mesma classe.
  • type: Utilizada para modificar o tipo da tag input. Ex: Quando utilzada no input pode alternar entre botão ou caixa de texto.
  • src: Indica a source ou fonte da imagem a ser atrelada ao site.
  • width: Pode ser usado ao inserir imagens no site, ele modifica a largura da imagem.
  • href: Utilizada unicamente com a tag "a", referencia o link.
  • target: Também utilizada unicamente com a tag "a", especifica o modo para abrir o link;
  • title: Define um título para um elemento. Ex.: Pode atribuir um titulo ao link.
  • name: Designa um nome para o elemento. Ex.: Define um nome para cada informação de um formulário
  • action: Indica o link em que o formulário será enviado.
  • method: Define qual método será utilizado pra enviar as informações do formulário para o servidor.
  • autocomplete: Define se o usuário poderá preencher os dados com as informações salvas no navegador.
  • onsubmit: Os atributos começados com on se tratam de eventos, neste caso ele realiza uma ação após o formulário ser enviado.
